# Formatos de exportacao

- JSON: projeto completo, cenarios, catalogos e premissas.
- CSV: lista de materiais em formato tabular.
- XLSX: lista de materiais para planilha.
- PDF relatorio: resumo de geometria, areas, materiais, custos e avisos.
- PDF tecnico: desenhos SVG preliminares convertidos para PDF.
- TXT RFQ: pedidos de cotacao em portugues.

Os PDFs sao gerados no navegador.

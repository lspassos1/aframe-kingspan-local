# Limitacoes

Esta ferramenta nao substitui:

- projeto arquitetonico;
- projeto estrutural;
- ART/RRT;
- aprovacao municipal;
- sondagem de solo;
- calculo de fundacoes;
- verificacao completa de vento;
- detalhamento de ligacoes metalicas;
- validacao tecnica do fornecedor dos paineis.

Os precos seed podem ficar desatualizados. O app mostra alerta quando a data da cotacao ultrapassa a validade configurada.
